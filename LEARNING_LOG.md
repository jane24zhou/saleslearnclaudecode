### What I built today
Set up the project from scratch! The repo now has:

### What I learned
- **HTML** gives a webpage its *structure* — it's like the frame of a house 🏠
- **CSS** makes things *look good* — like the paint and furniture 🎨
- **JavaScript** makes things *interactive* — like the light switches and doors ⚡
- **localStorage** is a way to save data in the user's browser without needing a database!
- `git` and GitHub let you save your code and share it with the world

### What was hard
- Understanding the difference between HTML, CSS, and JavaScript at first felt overwhelming
- Figuring out how to make data *persist* (stay saved when you close and reopen the app) — turns out localStorage handles this simply!
- Learning that every `<div>` in HTML needs to be closed with `</div>` 😅

### Questions I asked (and what I learned)
- *"What is the difference between a function and a variable in JavaScript?"*
  → A **variable** stores a value (like a box). A **function** does something (like a machine that can run repeatedly).
- *"Why do I need CSS if I can just put styles directly in HTML?"*
  → Separating CSS into its own file makes it easier to change the look of the whole site from one place.
- *"What does `localStorage.setItem` do?"*
  → It stores a small piece of data in the user's browser, kind of like a sticky note that survives the browser being closed.
